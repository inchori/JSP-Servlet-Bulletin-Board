# JSP-Servlet-Bulletin-Board

Try to develop a bulletin board by JSP/Servlet, Apache, OracleDB.

## Motivation

Review Object-oriented Paradigm.

## Dev Environment

* Java Version : Java Version 1.8.0_181
* Server : Apache Tomcat v8.5
* DataBase : Oracle DB 18c Express Edition

## Source & Link

* 나동빈 JSP 게시판 만들기 강좌
* https://https://youtu.be/wEIBDHfoMBg

## Difference

* JSP --> JSP/Servlet (MVC)
* MySQL --> Oracle DB
* JDBC --> Use Connection Pool

## Installation

1. Import Project
2. Server Setting
3. Run As to ``index.jsp``
